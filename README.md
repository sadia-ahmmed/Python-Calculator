# Python-Calculator

This is a Basic Python Calculator with a few additional trigonometric functionalities. It has **16 different options** that can perform mathematical operations. 

## Features
#### 1. Addition
#### 2. Subtraction 
#### 3. Multiplication 
#### 4. Division
#### 5. Modulus 
#### 6. Power
#### 7. Square root 
#### 8. x root over y 
#### 9. Degree to radian conversion
#### 10. Find sin
#### 11. Find cosine 
#### 12. Find tan
#### 13. Find inverse sin
#### 14. Find inverse cosine
#### 15. Find inverse tan
#### 16. Find factorial
